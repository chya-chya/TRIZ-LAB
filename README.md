# TRIZ-LAB

🧪 TRIZ LAB

TRIZ LAB은 문제 해결을 ‘결과’가 아닌 사고 과정 중심으로 공유하는 협업형 문제 해결 플랫폼입니다.
사용자는 문제를 등록하고, 다른 사용자들은 TRIZ(발명 문제 해결 이론) 프레임워크를 기반으로 구조화된 방식으로 해결을 시도합니다.

이 프로젝트는 단순한 Q&A 서비스가 아닌,
문제를 바라보는 관점과 사고 흐름을 축적하는 학습 공간을 목표로 합니다.

🚀 주요 기능

문제 등록 및 조회

TRIZ 기반 문제 해결 시도 작성

모순(Contradiction) 기반 사고 프레임 제공

발명 원리(40 Principles) 선택 및 기록

해결 시도에 대한 토론 및 피드백

좋아요를 통한 유용한 해결책 추천

🧩 서비스 핵심 컨셉

기존의 Q&A 서비스가 "정답"에 집중했다면,
TRIZ LAB은 다음과 같은 질문을 중심에 둡니다:

“문제 속 진짜 모순은 무엇인가?”

“포기하지 말아야 할 제약 조건은 무엇인가?”

“이 문제를 다른 관점에서 보면 어떨까?”

이를 위해 모든 답변은 TRIZ 구조화 폼으로만 작성할 수 있도록 설계했습니다.
